Textcat Desktop Client: Telesto

As of 2025-12-01, I decieded that textcat needed a new client, since the web client wasn't going to last long:

  • no support for voice calls
  • no support for desktop notifications

And without thoose features, textcat would be booooooring.

So the textcat telesto client was released, made by me using NodeJS and Electron (despite hating it).
